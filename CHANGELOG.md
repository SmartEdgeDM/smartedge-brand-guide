# Changelog

All notable changes to the SmartEdge Brand Guide are documented here.

---

## [1.1.0] — 2026-04-17

### Added
- `meta name="description"` tag for SEO
- Open Graph tags — `og:title`, `og:description`, `og:url`, `og:image`, `og:type`
- Twitter card meta tags for social sharing previews
- Favicon — SmartEdge lime slash mark embedded as SVG data URI
- Footer link to SmartEdgeDM.com
- `aria-label` attributes on all color swatches, logo variations, and business card previews
- `aria-hidden="true"` on decorative elements
- `README.md` with live site link, color table, and typography overview
- `LICENSE` — MIT license
- Fixed CSS typo: `flex-wrap: gap` corrected to `flex-wrap: wrap` in footer

### Fixed
- Color swatch copy buttons now include descriptive `aria-label` attributes for screen readers
- Removed unused `.hero-meta-item {}` empty ruleset

---

## [1.0.0] — 2025

### Initial Release
- Full brand identity system in a single HTML file
- Color palette — 5 brand colors with hex, CMYK, one-click copy
- Typography specimen — Montserrat and Georgia across all weights
- Logo system — 4 variations with clear space guidelines
- Business card preview — front and back with print specifications
- Voice & Tone guidelines — do's and don'ts
- Usage rules — 8 brand protection rules
- Scroll-triggered reveal animations
- Responsive mobile layout
