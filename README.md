## 🎨 Brand Colors

| Color | Hex | Use |
|-------|-----|-----|
| Almost Black | `#1A1A1B` | Primary — backgrounds, headings |
| Creamy White | `#F9F8F3` | Background — light surfaces, body text |
| Royal Azure Blue | `#0056B3` | Trust — H2/H3 headings, links |
| Cyber Lime | `#CCFF00` | Accent — CTA buttons, logo slash |
| Light Purple | `#B19CD9` | Secondary — icons, dividers |

## 🖋 Typography

- **Display / Headlines:** Montserrat Black (900)
- **Subheadings:** Montserrat Bold (700) in Royal Azure Blue
- **Body Copy:** Georgia Regular (400)
- **Labels / UI:** Montserrat Medium (500)
